YT2TikTok Auto Uploader

A private tool that downloads the owner's YouTube videos and posts one per day to the owner's TikTok account.

Private use only No user data Content Posting API

This web/desktop app automates posting: it temporarily saves a video on the server, uploads it to TikTok using TikTok’s Content Posting API, then deletes the file. Only the owner can authorize and use the app.
